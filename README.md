# OpenRouter AI Chatbot

A simple AI chatbot built using the MERN Stack and powered by OpenRouter API.
This application allows users to chat with an AI model through a clean and interactive web interface.

# Features
AI powered responses
Interactive chat interface
REST API backend
Secure API key using environment variables
Simple UI with HTML, CSS, and React
🛠 Tech Stack

Frontend

React.js
CSS

Backend

Node.js
Express.js

AI Integration

OpenRouter API

⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/athulall/openrouter-ai-chatbot.git
2️⃣ Install dependencies

Frontend

cd frontend
npm install

Backend

cd backend
npm install
🔑 Environment Variables

Create a .env file inside the backend folder.

OPENROUTER_API_KEY=your_api_key_here
▶️ Run the Project

Start backend server

npm start

Start frontend

npm run dev
